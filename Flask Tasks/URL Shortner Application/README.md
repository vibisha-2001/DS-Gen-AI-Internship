# URL Shortener Web Application

A simple URL Shortener built using Flask and SQLite.

## Features
- Shortens long URLs
- Redirects short URLs
- Stores URLs in database
- Shows URL history

## Technologies Used
- Python
- Flask
- SQLite
- HTML, CSS

## How to Run
python app.py
