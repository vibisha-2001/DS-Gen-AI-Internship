# URL Shortener Web Application

A simple web app built with **Python (Flask)** and **SQLite** to shorten long URLs and manage them per user.

## Features
- User **signup** and **login**  
- Generate **unique short URLs**  
- **Dashboard** to view and copy URLs  
- Flash messages for errors and notifications  

## Tech Stack
- **Backend:** Python, Flask, SQLAlchemy  
- **Database:** SQLite  
- **Frontend:** HTML, CSS, JavaScript  
- **Libraries:** validators
